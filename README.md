# Ethereum Smart Contracts Collection

Коллекция смарт-контрактов на Solidity для сети Ethereum.

## Контракты

### 1. Voting (Система голосования)

Децентрализованная система голосования, позволяющая создавать предложения и голосовать за них.

#### Основные возможности:
- Регистрация избирателей
- Создание предложений для голосования
- Голосование "за" или "против"
- Автоматическое отслеживание дедлайнов
- Прозрачность результатов

#### Основные функции:

```solidity
// Регистрация нового избирателя (только владелец)
registerVoter(address _voter)

// Создание предложения
createProposal(string memory _description)

// Голосование за предложение
vote(uint256 _proposalId, bool _support)

// Получение информации о предложении
getProposal(uint256 _proposalId)

// Исполнение предложения после окончания голосования
executeProposal(uint256 _proposalId)
```

#### Пример использования:

1. Владелец регистрирует избирателей
2. Зарегистрированный пользователь создает предложение
3. Избиратели голосуют в течение периода голосования (по умолчанию 7 дней)
4. После окончания срока предложение может быть исполнено

### 2. Crowdfunding (Краудфандинг платформа)

Платформа для сбора средств на проекты с автоматическим возвратом при неудаче.

#### Основные возможности:
- Создание проектов для сбора средств
- Взносы от спонсоров
- Автоматический возврат средств при недостижении цели
- Вывод средств создателем при успехе
- Комиссия платформы (2% по умолчанию)

#### Основные функции:

```solidity
// Создание нового проекта
createProject(
    string memory _title,
    string memory _description,
    uint256 _goalAmount,
    uint256 _durationDays
)

// Внесение средств в проект
contribute(uint256 _projectId) payable

// Вывод средств создателем (при успехе)
withdrawFunds(uint256 _projectId)

// Возврат средств спонсору (при неудаче)
refund(uint256 _projectId)

// Получение информации о проекте
getProject(uint256 _projectId)

// Получение вклада конкретного адреса
getContribution(uint256 _projectId, address _contributor)
```

#### Пример использования:

1. Создатель создает проект с целевой суммой и сроком
2. Спонсоры вносят средства
3. Если цель достигнута до дедлайна:
   - Создатель может вывести средства (минус комиссия платформы)
4. Если цель не достигнута после дедлайна:
   - Спонсоры могут вернуть свои средства

## Требования

- Solidity ^0.8.20
- Node.js (для тестирования)
- Hardhat или Truffle (для деплоя)

## Установка и деплой

### Вариант 1: Hardhat

```bash
npm install --save-dev hardhat
npx hardhat init
```

Скопируйте контракты в папку `contracts/` и запустите компиляцию:

```bash
npx hardhat compile
```

### Вариант 2: Remix IDE

1. Откройте [Remix IDE](https://remix.ethereum.org)
2. Создайте новый файл и скопируйте код контракта
3. Скомпилируйте контракт
4. Разверните в выбранной сети
