# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-11-07

### Added

#### Smart Contracts
- Solidity: 10 production-ready contracts (ERC-20, ERC-721, DeFi, DAO)
- Vyper: ERC-20 token implementation
- Rust: Solana Anchor counter program
- Move: Aptos fungible token contract
- Haskell: Plutus vesting validator for Cardano

#### Applications
- TypeScript: React DApp with Wagmi v2 and Viem
- Python: Web3.py CLI tools for wallet management
- Go: Ethereum RPC client with go-ethereum
- Java: Web3j enterprise integration example
- Swift: iOS/macOS wallet SDK

#### Utilities
- C++: Cryptographic algorithms (Keccak-256, Merkle trees)
- Bash: Node deployment and automation scripts
- AssemblyScript: WebAssembly modules for Web3

#### Frontend
- HTML/CSS/JS: Professional landing page

#### Documentation
- Comprehensive README with badges
- CONTRIBUTING.md guidelines
- CODE_OF_CONDUCT.md
- SECURITY.md policy
- Architecture documentation
- Deployment guides
- FAQ
- Russian translation

#### Testing
- Solidity: ERC-20 token tests
- Python: Wallet manager unit tests
- Rust: Counter program tests

#### CI/CD
- GitHub Actions workflows for automated testing
- Lint workflow for code quality
- Dependabot for dependency updates

#### Configuration
- .editorconfig for consistent code style
- .env.example files for all projects
- Enhanced .gitignore patterns

### Changed
- Reorganized repository structure for multi-language support
- Moved Solidity contracts to dedicated directory
- Updated root README to English

### Removed
- Duplicate root-level Hardhat configuration
