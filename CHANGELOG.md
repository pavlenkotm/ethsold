# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-11-05

### Added

#### Smart Contracts
- **Voting Contract** - Децентрализованная система голосования
  - Регистрация избирателей
  - Создание предложений
  - Голосование за/против
  - Автоматические дедлайны
  - Исполнение предложений

- **Crowdfunding Contract** - Платформа краудфандинга
  - Создание проектов
  - Внесение средств спонсорами
  - Автоматический возврат при неудаче
  - Вывод средств при успехе
  - Настраиваемая комиссия платформы (2% по умолчанию)

- **SimpleNFT Contract** - NFT контракт (ERC721-подобный)
  - Минт NFT с оплатой и бесплатно
  - Передача токенов
  - Система approvals
  - Управление supply
  - Вывод средств

#### Development Infrastructure
- **Hardhat Configuration** - Полная настройка для development
  - Поддержка Sepolia testnet
  - Поддержка Ethereum mainnet
  - Настроенный оптимизатор
  - Etherscan верификация

- **Package.json** - Зависимости и скрипты
  - NPM скрипты для компиляции, тестов и деплоя
  - Все необходимые dependencies

#### Deployment Scripts
- `deploy-voting.js` - Деплой контракта Voting
- `deploy-crowdfunding.js` - Деплой контракта Crowdfunding
- `deploy-nft.js` - Деплой контракта SimpleNFT
- `deploy-all.js` - Деплой всех контрактов
- Автоматическое сохранение адресов контрактов

#### Example Scripts
